# Roost

A shared life dashboard for couples. Shopping lists, expenses, chores, and a live activity feed — synced in real time between two Macs via Supabase.

Built with Electron, React, TypeScript, and Supabase.

**Read [`NORTH_STAR.md`](./NORTH_STAR.md) for the full product vision, roadmap, and session log.**

## Running locally

```bash
npm install
npm run dev
```

Requires a `.env.local` file with your Supabase credentials (see `.env.example`).
